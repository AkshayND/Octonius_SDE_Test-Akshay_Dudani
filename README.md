# Octonius_SDE_Test-Akshay_Dudani
To Do Application with MEAN Stack

## Steps to Run:
1. Download the folder
2. Extract it.
3. Open Command Prompt and Go to the extracted folder.
4. Run **`npm install`**
5. Run **`npm start`**
6. Go to http://localhost:3000/
7. Enjoy The To-Do App.
